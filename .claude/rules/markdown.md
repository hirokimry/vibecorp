# 📝 Markdown フェンスコードブロック規約

> [!IMPORTANT]
> Markdown のフェンスコードブロックには **必ず言語指定** を付ける。
> 言語指定なしのコードブロックは禁止。
> `SKILL.md` / エージェント定義 / ルール本文のテンプレ文字列も対象。

本ルールは全 `.md` ファイルに常駐適用する。コードブロックの言語指定を強制し、シンタックスハイライト・LLM 行内解釈・コード断片の取り違えを防ぐ。

## 🎯 対象範囲

| 対象 | 適用 |
|------|------|
| プロジェクト直下 `.md` | ✅ 適用する |
| `.claude/rules/**/*.md` | ✅ 適用する |
| `.claude/agents/**/*.md` | ✅ 適用する |
| `skills/**/SKILL.md` | ✅ 適用する |
| `docs/**/*.md` | ✅ 適用する |
| `README.md` | ✅ 適用する |
| プロンプト本文・テンプレ文字列内のコードブロック | ✅ 適用する |

## ✅ 言語指定の例

| 用途 | 言語指定 |
|------|---------|
| シェル | ` ```bash` / ` ```sh` |
| 平文・ログ・ツリー | ` ```text` |
| マークダウン本体 | ` ```markdown` |
| YAML | ` ```yaml` |
| JSON | ` ```json` |
| TypeScript / JavaScript | ` ```typescript` / ` ```javascript` |
| Python | ` ```python` |
| diff | ` ```diff` |

## ❌ 禁止パターン

- ❌ 言語指定なし（` ``` ` のみ）
- ❌ `SKILL.md` / エージェント定義 / ルール本文内のテンプレ文字列で言語指定を省略する
- ❌ プロンプト出力例コードブロックで言語指定を省略する
