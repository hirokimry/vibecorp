# vibecorp プロダクト仕様書

> このドキュメントはプロダクトの公式仕様を定義する Source of Truth です。

## 概要

vibecorp は AI エージェントを組織化してプロダクト開発を回すプラグイン。3段階のプリセット（minimal / standard / full）で段階的に機能を拡張する。

## プリセット別機能表

| 機能 | minimal | standard | full |
|------|---------|----------|------|
| コードレビュー自動化 | ✓ | ✓ | ✓ |
| PR 自動作成 (/ship) | ✓ | ✓ | ✓ |
| ドキュメント整合チェック | — | ✓ | ✓ |
| レビュー指摘の規約反映 | — | ✓ | ✓ |
| セッション知見の吸い上げ | — | ✓ | ✓ |
| Role Agent（C-suite） | — | CTO, CPO | 全 C-suite + 分析員 |
| 自律改善 (/autopilot) | — | — | ✓ |
| 並列 ship (/ship-parallel) | — | — | ✓ |
| スパイクループ (/spike-loop) | — | — | ✓ |
| コンテナ隔離 | 不要 | 不要 | **必須** |
| Docker 前提条件 | 不要 | 不要 | **必須** |
